Como Criar um Repositório no GitHub

  1. Acesse o GitHub: Faça login na sua conta em github.com.
  2. Crie um Novo Repositório: Clique no botão verde "New" no topo da página inicial.
  3. Preencha as Informações:
    • Nome do Repositório: Escolha um nome descritivo para o projeto.
    • Descrição: Insira detalhes sobre o objetivo do repositório (opcional).
    • Visibilidade: Escolha entre Público ou Privado.
  4. Adicione um README: Marque a opção "Add a README file" para incluir um arquivo inicial.
  5. Finalize: Clique no botão verde "Create Repository".

Como Postar Arquivos no Repositório

  1. Inicializar o GIT no projeto: Utilizando o Prompt de Comando ou terminal do Vscode você deve digitar o comando abaixo: 
    git init

    Lembrando que você deve abrir o diretório onde seu projeto está salvo localmente para começar a postar.

  2. Adicionar os arquivos ao repositório: 
    git add .
  3. Criar o primeiro commit: 
    git commit -m "Primeiro commit"
  4. criar o arquivo main: O comando git branch -M main é frequentemente usado para renomear a branch padrão (geralmente chamada de master) para main, pois muitos projetos atualmente adotam main como padrão. 
    git branch -M main
  5. Conectar o repositório local ao GitHub: 
    git remote add origin *URL do seu repositório do GitHub*
  6. Enviar os arquivos para o GitHub: 
    git push -u origin main

Pronto, agora basta verificar se todos os arquivos foram postados corretamente no seu repositório do GitHub.
